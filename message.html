Commandes du Bot Discord

a!say [canal] [message]

Envoyer un message dans un canal spécifié.

Usage: a!say [canal] [message]

a!dm [membre] [message]

Description: Envoyer un message privé à un membre.

Usage: a!dm [membre] [message]

a!anonymous [canal]

Description: Envoyer un message anonyme dans un salon.

Usage: a!anonymous [canal]
