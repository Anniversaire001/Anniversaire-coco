Commandes du Bot Discord

a!setrecurringmessage

Définir le message récurrent.

Usage: a!setrecurringmessage

a!setrecurringinterval

Définir l'intervalle en minutes.

Usage: a!setrecurringinterval

a!setrecurringchannel #channel

Définir le canal où le message sera envoyé.

Usage: a!setrecurringchannel #channel

a!getrecurringstatus

Afficher les paramètres récurrents actuels.

Usage: a!getrecurringstatus

a!deleterecurringmessage

Supprimer le message récurrent.

Usage: a!deleterecurringmessage
